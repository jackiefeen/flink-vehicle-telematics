FLINK

# Start Flink
./bin/start-cluster.sh  

# Stop Flink
./bin/stop-cluster.sh

Create a new Maven project:
mvn archetype:generate                               \
      -DarchetypeGroupId=org.apache.flink              \
      -DarchetypeArtifactId=flink-quickstart-java      \
      -DarchetypeVersion=1.3.2
	  
For the execution open two terminals:
1) Repo folder (contains the maven project)
2) flink folder where flink is intalled
	  
In the Repo folder (the maven project) where the pom.xml file is located:
mvn clean install -Pbuild-jar

Then: in the flink folder:

####
./bin/flink run -c master2018.flink.VehicleTelematics /home/jackie/flink-vehicle-telematics/flinkProgram/target/flink-1.jar file:///home/jackie/flink-1.3.2/data/testdata file:///home/jackie/flink-1.3.2/data/outtest
